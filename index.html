Conductivity Meter

A tool for measuring the electrical conductivity, of various solutions.

View Full Picture

The aim of this tool measuring the electrical conductivity of various solutions using an AC measurement cell. The circuit starts with a DC power supply of -12V, +12V, and GND, followed by an oscillator that generates a signal with a voltage of 2.15V and a frequency of 0.7KHz using the operational amplifier TL081. Next, the signal is adapted and connected to the probe (a measurement cell with dimensions 1 cm and a distance of 1 cm between the probes). The current is converted to voltage using a current-to-voltage converter, and then peak detection is performed using a peak detector. The signal is filtered using an active low-pass filter, and finally, it is connected to a micro-controller PIC16F887A to display the results on a 16*2 I2C LCD screen. The device calculates and displays the conductivity value of the solution.

Gas and Flame Alarm

Gas and flame alarm based on Arduino.

View Full Video

The Gas and Flame Alarm Based on Arduino is a safety system designed to detect hazardous gases and flames. Using an Arduino microcontroller, the system integrates a gas sensor to monitor flammable gases and a flame sensor to detect fire.

Key Features:
Gas Detection: Monitors for gases like methane and LPG, triggering alerts if dangerous levels are detected. Flame Detection: Identifies flames through infrared radiation and provides an early warning. Alerts: Activates a buzzer and LED indicators for immediate notification. Display: Features an LCD screen for real-time status updates.
Applications: Home safety, Industrial environments, Laboratories...

This project provides real-time monitoring and alerts to enhance safety in various settings.
